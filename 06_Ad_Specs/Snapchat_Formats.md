# Snapchat Ad Formats: Second Chair

> **Platform Focus:** Full-screen vertical experiences for younger demographics.
> 
> **Key Insight:** Snapchat users expect fast, visual-first content.

---

## Ad Copy Fields

| Field | Character Limit | Notes |
|-------|-----------------|-------|
| **Headline** | 34 characters | Top of ad |
| **Brand Name** | 25 characters | "Claim Justice Now" |
| **CTA Button** | Standard options | Swipe Up action |

### CTA Button Options
- Learn More
- Sign Up
- Get Now
- Apply Now
- View

## Single Image/Video Ads

### Snap Ads
| Attribute | Specification |
|-----------|---------------|
| **Dimensions** | 1080 x 1920px (9:16) |
| **Aspect Ratio** | 9:16 (full screen) |
| **Duration (Video)** | 3-180 seconds (3-10 sec optimal) |
| **File Size** | Max 1GB (video), 5MB (image) |
| **Format** | MP4, MOV (video) / JPG, PNG (image) |
| **Audio** | Optional but recommended |

## Creative Best Practices

### What Works
- Bold, attention-grabbing visuals
- Text overlays for context
- Quick cuts and motion
- Native Snapchat aesthetic
- Authentic, non-polished

### What Doesn't Work
- Horizontal content
- Slow pacing
- Heavy text without visuals
- Corporate/formal tone

---

## Second Chair Specific Requirements

### Compliance
- Disclosure visible on screen
- No injury imagery
- Third-person claims

### Targeting
- Age: 18+ (Snapchat skews younger)
- Geographic: State-based
- Interest: News, automotive

### Demographic Note
Snapchat audience skews 18-34. Messaging should resonate with younger MVA victims (rideshare accidents, new drivers).

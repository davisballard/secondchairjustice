# Meta Ad Formats: Second Chair

## Special Ad Category Reminder

All Second Chair ads on Meta MUST be created under **Special Ad Categories** (Legal Services).

This limits:
- Targeting options (broad only)
- Audience features (no detailed interests, no lookalikes)
- Some placement options

---

## Feed Ads

### Single Image
| Attribute | Specification |
|-----------|---------------|
| **Dimensions** | 1080 x 1080px (1:1) — Recommended |
| **Alternative** | 1200 x 628px (1.91:1) — Landscape |
| **File Size** | Max 30MB |
| **Format** | JPG or PNG |

### Carousel
| Attribute | Specification |
|-----------|---------------|
| **Dimensions** | 1080 x 1080px (1:1) per card |
| **Cards** | 2-10 images |
| **File Size** | Max 30MB per image |

### Video
| Attribute | Specification |
|-----------|---------------|
| **Dimensions** | 1080 x 1080px (1:1) or 1080 x 1920px (9:16) |
| **Duration** | Up to 240 minutes (15-60 seconds recommended) |
| **File Size** | Max 4GB |
| **Format** | MP4 or MOV |

---

## Stories/Reels

### Stories
| Attribute | Specification |
|-----------|---------------|
| **Dimensions** | 1080 x 1920px (9:16) |
| **Duration** | Up to 60 seconds |
| **Safe Zones** | Top 14% and bottom 20% for UI elements |

### Reels
| Attribute | Specification |
|-----------|---------------|
| **Dimensions** | 1080 x 1920px (9:16) |
| **Duration** | Up to 90 seconds |
| **Audio** | Sound-on recommended |

---

## Text Limits

| Element | Character Limit | Recommendation |
|---------|-----------------|----------------|
| **Primary Text** | 125 characters | Before "See More" truncation |
| **Headline** | 27 characters | Visible in feed |
| **Description** | 27 characters | Below headline |
| **CTA Button** | Standard options | "Learn More" recommended |

---

## Second Chair Specific Requirements

### Safe Zones for Disclosures
- Footer area for "This is an advertisement"
- Must be readable at ad size
- Consider mobile visibility

### Visual Compliance
- No blood/injury imagery
- No distressed faces
- Recovery/hope imagery only
- See `Banned_Imagery.md`

### Copy Compliance
- Third-person language only
- No "you" + injury
- See `Third_Person_Rules.md`

---

## Recommended Ad Formats

| Objective | Best Format | Notes |
|-----------|-------------|-------|
| **Lead Generation** | Single Image or Video | Direct, clear CTA |
| **Awareness** | Video (15-30 sec) | Emotional, story-based |
| **Retargeting** | Carousel | Multiple case types |

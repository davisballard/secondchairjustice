## Text Overlay Weapons

### ðŸ”² SEMI-TRANSPARENT BACKGROUND BLOCK

**The most reliable overlay for mixed backgrounds.** Ensures text is readable regardless of image content.

| Style | Settings | When To Use |
|-------|----------|-------------|
| **Dark Scrim** | Black @ 40-70% opacity | White/light text over any image |
| **Light Scrim** | White @ 50-70% opacity | Dark text over busy/dark images |
| **Brand Color Scrim** | Your color @ 70-85% opacity | Consistent brand presence |
| **Gradient Fade** | 0%â†’70% opacity directional | Text at one edge of image |

**Best Practices:**
- Extend block at least 20px past text on all sides
- Round corners slightly (4-8px radius) for modern feel
- Match block shape to text shape (horizontal block for horizontal headline)
- Use consistent opacity across all ads in campaign

**Second Chair Application:** Use **dark scrim @ 60% opacity** behind white text as your default. This works on 90% of image backgrounds without obscuring the image entirely.

---

### ðŸŒ‘ DROP SHADOW

**Subtle text separation without blocking the image.** Works best when image background is relatively uniform.

| Style | Settings | When To Use |
|-------|----------|-------------|
| **Subtle Lift** | 2px offset, 4px blur, 30% black | Clean separation, minimal impact |
| **Strong Shadow** | 4px offset, 8px blur, 50% black | Busy backgrounds, need more pop |
| **Glow Effect** | 0px offset, 8px blur, centered | Dark text on varied background |

**Best Practices:**
- Shadow color should be darker version of background, not pure black
- Keep blur soft (4-8px) to avoid "dated" look
- Avoid multiple shadows or rainbow shadows
- Test on mobileâ€”shadows can blur together at small sizes

**Second Chair Application:** Use **subtle lift shadow** (2px, 4px blur) as secondary treatment when image is relatively uniform. Don't rely on shadow alone for busy imagesâ€”add a scrim.

---

### âœï¸ TEXT STROKE/OUTLINE

**High contrast treatment for maximum legibility.** Can look dated if overdone.

| Style | Settings | When To Use |
|-------|----------|-------------|
| **Thin Outline** | 1-2px contrasting color | Subtle edge definition |
| **Bold Outline** | 3-4px contrasting color | Maximum contrast, "poster" look |
| **Knockout** | White text, dark outline (or reverse) | When you need guaranteed readability |

**Best Practices:**
- Keep stroke width proportional to text weight (thin stroke on thin font, thick on bold)
- Use complementary or contrasting color, not similar
- Avoid colored strokes that compete with brand colors
- Test at actual sizeâ€”strokes can blur or disappear on mobile

**Second Chair Application:** Use **thin 2px outline** as emergency treatment when image is extremely busy and you can't add a scrim. Prefer scrim over stroke for professional appearance.

### ðŸŽ¯ SOLID COLOR BLOCK/BANNER

**Maximum readability, maximum brand presence.** Fully hides image beneath text.

| Style | Settings | When To Use |
|-------|----------|-------------|
| **Full Banner** | 100% opacity brand color | Hero text, main CTA |
| **Accent Strip** | 100% opacity highlight color | Benefit callout, "No Fee" messaging |
| **Boxed CTA** | High-contrast button shape | Primary call-to-action |

**Best Practices:**
- Use when text is more important than image underneath
- Align to image edges or centerâ€”avoid "floating" randomly
- Ensure text color meets 4.5:1 contrast ratio on block color
- Works well for CTAs to visually separate from rest of creative

**Second Chair Application:** Use **solid brand color banners** for primary CTAs. Use **gold/yellow accent strips** for benefit messaging ("No Fee Unless You Win").

## Typography Hierarchy Formula

### The 60-30-10 Rule for Text

| Element | % of Visual Weight | Font Treatment |
|---------|-------------------|----------------|
| **Headline** | 60% | Display/Bold font, largest size, boldest weight |
| **Sub-headline/Support** | 30% | Sans-serif, 50-60% headline size, medium weight |
| **CTA/Fine Print** | 10% | Sans-serif, smallest, distinct via color/button |

### Sizing Guidelines (For 1080x1080 Meta Ads)

| Element | Size Range | Notes |
|---------|------------|-------|
| **Headline** | 60-90pt | Must be readable at thumbnail (150x150px) |
| **Sub-headline** | 36-48pt | Readable on mobile feed |
| **Body/Benefit** | 24-32pt | Minimum for mobile legibility |
| **CTA Button Text** | 28-36pt | Large enough to tap, clear enough to read |
| **Fine Print** | 14-18pt | Legal/disclaimer only |

## Platform-Specific Adjustments

### Meta Feed Ads (1:1, 4:5)

- **Headline:** 72-90pt display font
- **Overlay:** Dark scrim @ 60%
- **Text amount:** 5 words or fewer in headline
- **Position:** Upper 1/3 to avoid thumb-zone overlap

### Meta Stories/Reels (9:16)

- **Headline:** 80-100pt (larger due to taller format)
- **Position:** Middle 60% (avoid top nav, bottom UI)
- **Overlay:** Full-width banner or gradient fade from bottom
- **Animation:** Consider animated text for higher engagement

### Google Display Network

- **Smaller sizes (300x250, 320x100):** Extra-bold fonts only
- **Contrast:** Maximumâ€”small formats need high contrast
- **Text:** Keep to 2-3 words per size, CTA only if fits

### YouTube Thumbnails

- **Size:** 150pt+ for headline text
- **Font:** Thick bold display only (Impact, Bebas Neue)
- **Overlay:** Strong outline OR bold scrimâ€”must read at sidebar size (120x68px)

## A/B Testing Priority

Based on research, test these in order of potential impact:

### 1. Font Weight (+15-25% potential)
Test **bold display font** vs **regular weight sans-serif** for headlines
- Hypothesis: Bolder = higher scroll-stop rate

### 2. Overlay Style (+18-27% potential)
Test **semi-transparent scrim** vs **no overlay with shadow** vs **solid color block**
- Hypothesis: Scrim provides best balance of visibility + image impact

### 3. Text Amount (+20-40% potential)
Test **3-word headline** vs **6-word headline** vs **full sentence**
- Hypothesis: Fewer words = faster comprehension = higher action

### 4. Case Treatment (+10-15% potential)
Test **ALL CAPS** vs **Title Case** vs **sentence case** for headlines
- Hypothesis: All caps performs better for short urgent messages

### 5. Font Family (+5-10% potential)
Test **geometric sans** (Bebas Neue) vs **humanist sans** (Roboto) for headlines
- Hypothesis: May vary by audienceâ€”test both

## What NOT To Do

| Mistake | Why It Fails |
|---------|--------------|
| Script/cursive fonts for headlines | Unreadable on mobile, slows comprehension |
| Light/thin font weights | Disappears on busy backgrounds, hard to read |
| More than 2 font families | Visual chaos, unprofessional |
| Rainbow text or gradient fills | Distracting, reduces legibility |
| Text over face/key image elements | Obscures important visual information |
| Justified text alignment | Creates uneven spacing, hard to read |
| Yellow text on white background | Fails contrast ratio, invisible |
| Decorative fonts for body copy | Exhausting to read, kills comprehension |
| Matching text color to image colors | No contrast = invisible text |
| Text in bottom 15% (Stories/Reels) | Covered by platform UI |

## Quick Reference: Text Legibility Checklist

Before exporting any ad creative:

- [ ] Headline uses bold/display font at 60pt+
- [ ] Body text uses clean sans-serif at 24pt+
- [ ] Text has overlay treatment (scrim, shadow, or outline)
- [ ] Contrast ratio meets 4.5:1 minimum
- [ ] Maximum 2 font families used
- [ ] All-caps reserved for headlines/CTAs only
- [ ] Text avoids platform UI zones (top/bottom on Stories)
- [ ] Readable at thumbnail size (150x150px or smaller)
- [ ] CTA text is distinct (color, button, or weight)
- [ ] Total text coverage under 20% of image area
